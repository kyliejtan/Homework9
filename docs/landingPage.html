Summary: Latitude vs. X

The goal of this project was to examine how climate changes with latitude across our planet. Our first step in accomplishing this goal was to collect temperature, humidity, percent of cloud cover, and wind speed data as well as the latitude and longitude coordinates from over 500 cities around the world. We queried the OpenWeatherMap API for this data using the names of 546 randomly selected city, and subsequently built a Pandas data frame that would allow us to aggregate the collected data and begin plotting each group of climate data as a function of latitude using Matplotlib.

This site presents our findings across individual pages that depict the observed trends of each climate variable in respect to latitude, as well as on a comparison page that displays all of our plots together. Our source data is also available on its own page.
